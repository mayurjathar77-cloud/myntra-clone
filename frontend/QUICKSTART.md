# 🚀 Quick Start Guide

## Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Run the Application
```bash
npm run dev
```

### Step 3: Open in Browser
Visit: `http://localhost:5173`

## ✅ That's it! Your app is running!

### What You Can Do Now:
- ✅ Browse products
- ✅ Add items to bag
- ✅ Complete checkout (Demo Mode)
- ✅ View orders
- ✅ Manage wishlist

### Optional: Enable Real Payments

1. Get Razorpay Test Key from https://razorpay.com/
2. Update `data/razorpayConfig.js`:
   ```javascript
   KEY_ID: "rzp_test_YOUR_KEY_HERE"
   ```
3. Test with card: 4111 1111 1111 1111

### Optional: Start Backend Server

```bash
npm run server
```

Server runs on: `http://localhost:8080`

## 🎉 Enjoy Shopping!

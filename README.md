# 🛍️ Myntra Clone

A modern e-commerce web application built with React, Redux Toolkit, and Express.js backend.

![React](https://img.shields.io/badge/React-19.2.0-blue)
![Redux](https://img.shields.io/badge/Redux_Toolkit-2.11.2-purple)
![Node.js](https://img.shields.io/badge/Node.js-Express-green)

## ✨ Features

- 🛒 Shopping cart and wishlist
- 💳 Razorpay payment integration
- 📦 Order management
- 👤 User profile with loyalty points
- 📧 Email notifications
- 🔍 Search and filtering
- 📱 Responsive design

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- npm

### Installation & Run

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
Runs on: `http://localhost:5173`

**Backend:**
```bash
cd node-backend
npm install
npm start
```
Runs on: `http://localhost:3000`

## 🌐 Deploy to GitHub Pages

```bash
cd frontend
npm run build
npm run deploy
```

## 🛠️ Tech Stack

- **Frontend:** React, Redux Toolkit, Bootstrap, Vite
- **Backend:** Express.js, Razorpay SDK, Nodemailer

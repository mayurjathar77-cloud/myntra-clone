# Manual GitHub Deployment Commands
# For: https://github.com/mayurjathar77-cloud

## Step 1: Create New Repository on GitHub
1. Go to: https://github.com/mayurjathar77-cloud
2. Click "+" icon → "New repository"
3. Repository name: myntra-clone
4. Description: Full-stack e-commerce application with React, Redux, and Razorpay
5. Choose Public
6. DO NOT initialize with README
7. Click "Create repository"

## Step 2: Run These Commands in Terminal

# Initialize Git
git init

# Add all files
git add .

# Create first commit
git commit -m "Initial commit: Myntra clone full-stack e-commerce application"

# Set main branch
git branch -M main

# Add remote repository
git remote add origin https://github.com/mayurjathar77-cloud/myntra-clone.git

# Push to GitHub
git push -u origin main

## Step 3: Verify
Visit: https://github.com/mayurjathar77-cloud/myntra-clone

## Future Updates
git add .
git commit -m "Your update message"
git push

## Deploy Live (After GitHub Push)

### Frontend on Vercel:
1. Visit: https://vercel.com
2. Import repository: mayurjathar77-cloud/myntra-clone
3. Root Directory: frontend
4. Build Command: npm run build
5. Output Directory: dist
6. Deploy

### Backend on Render:
1. Visit: https://render.com
2. New Web Service
3. Connect repository: mayurjathar77-cloud/myntra-clone
4. Root Directory: node-backend
5. Build Command: npm install
6. Start Command: npm start
7. Deploy
